Install and setup config

``` sudo apt-get install snmp-mibs-downloader sudo download-mibs # Finally comment the line saying "mibs :" in /etc/snmp/snmp.conf sudo vi /etc/snmp/snmp.conf ```

Enumerating SNMP

``` snmpwalk -v <1/2c/3> -c .1 #Enum all snmpwalk -v <1/2c/3> -c NET-SNMP-EXTEND-MIB::nsExtendObjects #Get extended snmpwalk -v <1/2c/3> -c NET-SNMP-EXTEND-MIB::nsExtendOutputFull #Get extended full ```

Filter result with grep

``` grep -i "trap" #Get private string grep -i "login\|fail" #Get login grep -i "user\|pass" #Get user&pass grep -E -o "\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,6}\b" #Get email grep -i "hrSWInstalledName" #Get installed program grep -i "NET-SNMP-EXTEND-MIB" #Get extended ```
